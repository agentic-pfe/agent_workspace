# Orcheeos Brand Usage Guidelines

## Logo Lockups
- **Primary Lockup**: Horizontal logo (logo.svg) with "Orcheeos" in Inter 700, letter-spacing: 0.1em
- **Icon-Only**: Use icon from logo.svg for favicons, app icons, and small UI contexts
- **Vertical Lockup**: Available upon request for print layouts
- **Clear Space**: Minimum padding = logo height / 4
- **Minimum Size**: 24px width for digital, 15mm for print

## Color Usage
- **Primary**: #3d87cc — UI controls, headers, key data points
- **Secondary**: #b75088 — Accent lines, secondary data, hover states
- **Accent**: #80cc20 — Alerts, active states, success indicators (use sparingly)
- **Neutral**: Use gradients from #f4f7f9 → #1e1e1e for depth

## Typography
- **Headings**: Inter 700, letter-spacing: -0.02em
- **Body & Code**: JetBrains Mono 500, line-height: 1.6
- **Code Blocks**: Always use JetBrains Mono, background: #1e1e1e, text: #f4f7f9
- **Contrast Compliance**: All text meets WCAG AA on neutral backgrounds

## Icon System
- Use icons from /icons/ directory
- All icons are 24x24px, stroke: 1.5px, no fill
- Use #80cc20 for active icons, #8a8b8c for disabled

## Design Tone
- Technical, cold, instrument-panel aesthetic
- Inspired by oscilloscopes: sharp lines, grid-aligned elements, monochrome depth
- Avoid gradients unless used as subtle depth layers
- Never use rounded corners on core UI elements

## Prohibited
- Do not rotate, skew, or recolor logo
- Do not use accent color (#80cc20) as background
- Do not use serif fonts
- Do not use drop shadows or glows
