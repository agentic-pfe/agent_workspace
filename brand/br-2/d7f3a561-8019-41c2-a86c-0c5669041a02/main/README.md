# NovaFlow Brand Guide

## Installation
```bash
pip install reportlab Pillow
```

## Files
- `novaflow-logo.svg` — Primary logo
- `novaflow-icon-set.svg` — Icon system
- `palette.json` — Color definitions
- `typography-spec.json` — Typography scale
- `brand_guide.md` — Human-readable guide

## Usage
Run `install_deps.sh` to install dependencies, then execute the brand guide generation script.
