## Course Description (150 words)

Artificial intelligence is reshaping every industry — from healthcare to finance, education to entertainment. Yet most courses assume you already know how to code. This one doesn't.

**Artificial Intelligence from Scratch** is a project-based, beginner-friendly course designed for absolute newcomers and tech enthusiasts alike. You'll start with the fundamental concepts: what AI is, how machines learn, and the difference between AI, ML, and deep learning. From there, you'll dive into Python programming tailored for AI, explore real datasets, and build your first machine learning models — including linear regression, decision trees, and neural networks.

By the second half of the course, you'll work on three capstone projects: a spam detection engine, an image classifier using convolutional neural networks, and an interactive chatbot powered by natural language processing. Along the way, you'll learn how to evaluate model performance, avoid bias, and deploy AI applications to the cloud.

Every module includes quizzes, coding challenges, and real-world case studies. By graduation, you'll have a GitHub portfolio of AI projects, a certificate of completion, and the confidence to pursue a career in artificial intelligence.

**Duration:** 8 weeks · **Skill Level:** Beginner · **Format:** Self-paced with live weekly Q&A sessions.

## Key Design Notes (for UX/UI Worker)

- **Primary Color Palette:** Deep navy (#0a0e27), Electric Blue (#3b82f6), Neon Purple (#8b5cf6), White (#ffffff)
- **Typography:** Use a clean sans-serif font (e.g., Inter or Space Grotesk) for headings, and a readable system font stack for body
- **Hero Layout:** Full-width background image with left-aligned text overlay, gradient overlay for readability
- **Card Design:** Dark glass-morphism cards for feature sections with subtle glow borders
- **Accessibility:** Ensure 4.5:1 contrast ratio on all text, proper heading hierarchy, and focus states on all interactive elements
