## Accessibility Notes

- All heading levels follow a logical hierarchy: H1 → H2 → H3
- All images include descriptive alt text for screen readers
- CTA labels use action-oriented verbs for clarity
- Color contrast ratios for primary UI elements (teal #0d9488 on dark backgrounds) exceed 4.5:1
