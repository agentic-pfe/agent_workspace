# UX Content Brief — Hello World Page

## Purpose
A minimal "Hello, World!" landing page. No images, no CTAs, no navigation.

## Target Audience
General — anyone testing or referencing a simple web page.

## Content
- **Heading (h1)**: "Hello, World!"
- No other text, images, or interactive elements needed.

## Design Tokens
- **Primary**: #1a1a2e (dark text)
- **Background**: #f0f0f5 (light off-white)
- **Typography**: system sans-serif stack
- **Layout**: Single heading centered vertically and horizontally via flexbox.
