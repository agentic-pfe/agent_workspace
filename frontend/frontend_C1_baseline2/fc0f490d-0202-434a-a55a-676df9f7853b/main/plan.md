# Plan - Alex Morgan Portfolio

## Files
1. **content_brief.json** - Design tokens: palette (slate-blue + warm-stone + coral), typography (Playfair Display + Inter), hero/features/testimonials copy
2. **index.html** - Semantic HTML5: header+nav, hero, projects grid (4 cards), skills bars, about+timeline, contact form, footer
3. **styles.css** - Mobile-first, fluid typography, alternating sections, gradients, shadows, hover effects, animations
4. **app.js** - Mobile menu, smooth scroll + active link tracking, skills bar fill on scroll (IntersectionObserver), contact form validation, testimonial carousel

## Key Design Decisions
- Palette: 3 hue families (slate-indigo, warm-stone sand, coral accent) — feels minimal but not monochrome
- Typography: Playfair Display headings, Inter body — strong hierarchy
- Layout: Max-width container, generous whitespace, grid projects
- Interactions: Hover overlays on projects, animated skill bars, smooth scroll spy
