# Forma Studio Website Project Plan

Files to Create:
1. home.html - Landing page with hero, services, featured work
2. about.html - Team, mission, values
3. work.html - Portfolio grid with project showcases
4. contact.html - Contact form and location info
5. styles.css - Shared design system and styles
6. app.js - Shared interactive behaviors
7. report.md - Design system documentation

Design System Focus:
- Modern, minimalist design for design agency
- Color palette: Deep blue, soft coral, neutral grays
- Typography: Sharp, geometric sans-serif
- Grid-based layout with clean, spacious design
- Responsive across mobile and desktop
- Consistent component styles (buttons, cards, sections)
